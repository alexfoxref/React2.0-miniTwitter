    ОБЯЗАТЕЛЬНОЕ ЗАДАНИЕ: 

        1) Создать проект, повторяя за видео.

        2) Написать условие, что если в данных, которые приходят от сервера, есть что-то кроме объектов - они игнорируются.

        (Не забывайте про иммутабельный state)
